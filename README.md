# Литера - бот для анализа сочинений ОГЭ

Бот проверяет сочинения по критериям ФИПИ с помощью GigaChat.

## Команды
- `/start` - приветствие
- `/help` - помощь
- `/at` - анализ сочинения (3 шага: тема → исходный текст → сочинение)
- `/criteria` - критерии оценивания
- `/cancel` - отмена

## Технологии
- Python 3.11
- Aiogram
- GigaChat API
